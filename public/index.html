How to Use

Creating Tickets

  1. Switch to the "Create Ticket" tab
  2. Describe your issue in the text area
  3. Select the appropriate priority level
  4. Optionally assign the ticket to someone
  5. Click "Create Ticket" to generate the ticket

Searching Tickets

  1. Switch to the "Search Tickets" tab
  2. Enter a natural language search query
  3. Optionally specify a team ID
  4. Click "Search Tickets" to find matching tickets

Writing Good Descriptions

  • Be specific: Include what you were doing when the issue occurred
  • Include steps: List the steps to reproduce the problem
  • Expected vs Actual: Describe what you expected vs what happened
  • Environment: Mention browser, OS, or relevant system info
  • Error messages: Include any error messages you saw

Search Examples

Find bugs assigned to user:
